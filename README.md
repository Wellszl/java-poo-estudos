# Java POO - Estudos

Repositório dedicado aos meus estudos do curso **Java COMPLETO - Programação 
Orientada a Objetos** do professor Nélio Alves (Udemy).

O curso cobre desde os fundamentos da linguagem Java até tópicos avançados como
Spring Boot, JPA/Hibernate e MongoDB.

## Tecnologias abordadas

- Java
- Spring Boot
- JPA / Hibernate
- JDBC
- MongoDB
- Maven
- Git e GitHub

## Objetivo

Consolidar fundamentos de programação orientada a objetos em Java, 
com foco em modelagem de sistemas, boas práticas e preparação para 
desenvolvimento backend com Spring Boot.

## Estrutura

O repositório está organizado por módulos do curso:

/modulo-01-fundamentos  
/modulo-02-estruturas  
/modulo-03-poo  

Cada módulo contém:
- exercícios práticos
- exemplos de código
- anotações relevantes

## Observação

Este repositório contém exercícios e exemplos guiados pelo curso.  
Projetos autorais baseados nesses aprendizados serão publicados em repositórios separados.
